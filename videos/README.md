# Appliance Videos

Drop your how-to videos in this folder using these exact filenames:

| Appliance        | Filename               |
|------------------|------------------------|
| Coffee Machine   | `coffee_machine.mp4`   |
| Washing Machine  | `washing_machine.mp4`  |
| Cooktop          | `cooktop.mp4`          |
| Dishwasher       | `dishwasher.mp4`       |
| Oven             | `oven.mp4`             |
| Microwave        | `microwave.mp4`        |

**Tips:**
- MP4 format (H.264) works best on phones and browsers.
- Keep each video under 100 MB (GitHub's per-file limit).
- After adding videos, commit and push to update the live site.
